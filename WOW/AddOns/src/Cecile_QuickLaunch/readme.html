Cecile Quick Launch : Read Me


Usage::

First you need to set-up a keybinding in the window settings to open the launcher, by default it's CTRL+SHIT+P.

When the window pop-up you could start typing to search.

You could press ENTER to select the first option or use TAB or SHIFT-TAB to navigate trough the results.

Alternatively you could use mouse wheel/click, but honestly I don't recommend it.

You could define Aliases and Blacklist items in the search options.

Examples:

Summon a random pet:
CTRL+SHIT+P pet SPACE rando ENTER

Dismount: CTRL+SHIT+P dism ENTER

Summon your second favorite mount:
CTRL+SHIT+P mou SPACE favo TAB TAB TAB ENTER

Summon your Soaring Skyterror:
CTRL+SHIT+P mou SPACE skyt ENTER

Open an AddOn configuration:
CTRL+SHIT+P cfg SPACE name ENTER

Open this AddOn configuration:
CTRL+SHIT+P cql ENTER
